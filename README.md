# opentofu-aws-example

豆腐の練習

## init

```shell
$ cp .envrc.sample .envrc
$ tofu init
```

## version manager

```shell
$ brew intall tofuenv
$ tofuenv list-remote
$ tofuenv install latest
$ tofuenv use <version>
$ tofu -v
OpenTofu v1.6.0-dev
on darwin_arm64
```
